Latency Pipeline Breakdown

1. Motion

User moves head. IMU sensors capture rotation & position.

~1ms

2. CPU

Game logic updates. Render thread submits draw calls.

~5-8ms

3. GPU

Geometry rendering, shading, and distortion correction.

~5-8ms

4. Scan-out

Display illuminates pixels row by row (Rolling Shutter).

~5ms

5. Photon

Light hits the retina. Brain perceives motion.

Instant
Target Motion-to-Photon Latency

< 20ms Total
