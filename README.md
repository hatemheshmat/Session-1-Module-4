## üîß Troubleshooting

- Ensure all files are extracted to the same folder
- Check that `background.svg` is in the same directory as `index.html`

### Progress not saving?
- Enable cookies/localStorage in your browser
- Check browser privacy settings

### Images not displaying?
- Verify the `assets/` folder is present
- Check browser console for errors (F12)
