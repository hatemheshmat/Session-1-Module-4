Mesh Optimization Checklist

Geometry & Topology
Poly Count: Within budget for object type (e.g., <1000 tris for props).
Clean Topology: No N-gons, T-vertices, or non-manifold geometry.
Backfaces: Hidden faces deleted (e.g., bottom of a crate).
Transforms & Scale
Scale: Reset to (1, 1, 1). Modeled in real-world units (Meters).
Pivot Point: Correctly placed (Base for props, Center for physics).
UVs & Materials
UV Maps: No overlapping UVs (unless intentional stacking).
Materials: Single material per mesh whenever possible.
Real-World Scale
VR relies on accurate scale for presence. If a chair is 2 meters tall, the user will feel like a child. Always model with 1 Unit = 1 Meter.
The "Read/Write" Trap
In Unity Import Settings, ensure "Read/Write Enabled" is UNCHECKED unless you are modifying the mesh via script at runtime. Leaving it on doubles memory usage.
