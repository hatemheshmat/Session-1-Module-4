Texture Streaming

The Problem
Without streaming, Unity loads all referenced textures into VRAM at full resolution, even if the object is a tiny dot on the horizon. This leads to Out of Memory (OOM) crashes.
The Solution
The Mipmap Streaming system analyzes the scene and only loads the specific mip level needed for the current screen size.

  • Close Object: Loads Mip 0 (Full Res)
  • Far Object: Loads Mip 4 (Low Res)
Result: You define a fixed "Memory Budget" (e.g., 512MB), and Unity ensures textures never exceed it.
