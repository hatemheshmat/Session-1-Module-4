Object Pooling

The Problem
Creating objects (Instantiate) allocates memory. Destroying them (Destroy) creates garbage. When the Garbage Collector runs to clean up, your game freezes for a few milliseconds. In VR, this causes nausea.
The Solution
Recycle everything. Instead of destroying a bullet when it hits a wall, disable it and put it back in a "Pool". When you need a new bullet, take one from the pool and enable it.
