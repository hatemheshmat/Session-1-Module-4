Mesh Compression

Reduce Build Size

Mesh Compression reduces the file size of your assets on disk (APK size) by using quantization algorithms to compress vertex positions and normals.

    Off
    No compression. Maximum precision. Use for modular walls/floors to prevent gaps.
    Medium
    Good balance. Minor artifacts. Suitable for most props and characters.
    High
    Aggressive compression. Use for organic shapes (rocks, trees) where precision matters less.

The Trade-off

This is lossy compression. It does not improve runtime memory (RAM) significantly, but it drastically reduces download size and load times.
