VFX Best Practices

Texture Sheet Animation
Instead of spawning 50 particles to create a smoke trail, use 1 particle that plays a flipbook animation (Texture Sheet). This drastically reduces overdraw and vertex count.
Simulation Space: Local
For effects attached to moving objects (like torches or weapons), set Simulation Space to Local. This saves the CPU from transforming every particle into World Space each frame.
Prewarm Looping Effects
For environmental effects like fire or waterfalls, enable Prewarm. This ensures the effect is fully formed when the scene loads, avoiding a performance spike and visual "pop-in".
