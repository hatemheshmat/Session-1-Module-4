Skinned Mesh Optimization

The Cost of Animation
Skinned Meshes are expensive. Every frame, the CPU must calculate bone positions, and the GPU must deform the mesh vertices based on those bones.
Combine Meshes
Don't use separate meshes for Head, Body, and Clothes. Combine them into a single Skinned Mesh. This reduces Draw Calls and simplifies the rig.
Pro Tip: If a character is just a statue and doesn't move, change the component from SkinnedMeshRenderer to a standard MeshRenderer. It's much cheaper.
