Texture Compression Formats

Feature ASTC (New) ETC2 (Old)
Quality High (Adaptive) Medium (Fixed)
Flexibility Block-based (4x4 to 12x12) Fixed 4 bits/pixel
Alpha Channel Excellent Support Separate/Split
Build Size Very Low Moderate
Recommendation
Always use ASTC for Meta Quest development. It offers the best balance of visual fidelity and memory footprint. ETC2 should only be used for legacy Android device support (pre-2016).
Understanding Block Sizes
ASTC 4x4
8.00 bits/pixel
Use for: UI, Normal Maps, Hero Assets
ASTC 6x6
3.56 bits/pixel
Use for: Standard Textures, Props
ASTC 8x8
2.00 bits/pixel
Use for: Distant Objects, Skyboxes
