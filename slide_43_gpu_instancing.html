GPU Instancing

Mass Replication
GPU Instancing tells the GPU to draw the same mesh thousands of times in a single draw call. Unlike Static Batching, these objects can move, rotate, and scale independently.
vs. Static Batching
  • Low Memory Usage (No mesh copying)
  • Dynamic Transforms (Can move)
  • Requires Identical Mesh & Material
Best For:
Forests, Grass, Asteroid Fields, Particles, Bullets, Crowd Characters.
1 Draw Call
Per-Instance Properties (Color Variation)
MaterialPropertyBlock props = new MaterialPropertyBlock();

foreach (var obj in objects) {
  props.SetColor("_Color", Random.ColorHSV());
  obj.renderer.SetPropertyBlock(props);
}
// Allows unique colors without breaking the batch!
