The Transparency Trap
Transparent materials (Alpha Blend) cause Overdraw. The GPU must render pixels behind the object and the object itself. Use Opaque materials whenever possible.
Enable GPU Instancing
Always check "Enable GPU Instancing" on your materials. This allows Unity to draw many copies of the same mesh in a single draw call, massively reducing CPU overhead.
