Material Best Practices

Reuse Materials
Batching only works if objects share the exact same material instance. If you have 100 crates, use 1 material, not 100 copies.
Limit Transparency
Transparent pixels cause Overdraw. The GPU must render the background, then the object. Avoid large glass windows or full-screen smoke effects.
Channel Packing
Don't use 3 separate textures for Metallic, Smoothness, and Occlusion. Pack them into the R, G, B channels of a single texture to save memory.
Use Mobile Shaders
The "Standard" shader is heavy. Use "Mobile/Diffuse" or "URP/Simple Lit" whenever possible. Unlit shaders are the fastest option for stylized games.
PRO TIP: GPU Instancing requires identical materials.
