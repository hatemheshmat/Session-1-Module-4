Memory Management

The GC Spike
When you allocate memory (create objects) and then discard them, the Garbage Collector (GC) must eventually run to clean up. This process freezes the CPU completely.
The Impact
In VR, you have 11ms per frame. A GC collection often takes 5-10ms. This guarantees a dropped frame, causing a visual stutter and instant motion sickness.
The Allocation Killers
LINQ (.Where, .Select, .ToList)
String Concatenation in Update
Foreach Loops (on some older compilers)
new List() inside loops
