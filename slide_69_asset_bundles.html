Asset Bundle Management

Break the APK Limit
Don't try to fit your entire 10GB game into the initial download. Use Asset Bundles to package content (Levels, Characters, Audio) separately and download them only when needed.
Use Addressables
Raw Asset Bundles are difficult to manage (dependencies, loading order). Use Unity's Addressables System. It handles memory management, reference counting, and loading automatically.
