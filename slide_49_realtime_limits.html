Real-time Lighting Limits

Shadow Casters
MAX 1
Only one Directional Light (the Sun) should cast shadows. Real-time point/spot light shadows are too expensive for mobile VR.
Pixel Lights
MAX 3
The number of important lights affecting a single object. Each additional pixel light adds a rendering pass.
Vertex Lights
MAX 4
Cheaper lights calculated per-vertex. They look lower quality but are much faster. Use for small dynamic lights (explosions, spells).
Why? Forward Rendering
Mobile VR uses Forward Rendering. This means for every pixel light touching an object, the object must be drawn again.
