Max Particles Limit: < 500
Hard limit on the number of particles alive at once. Keep this low. If you need more density, increase particle size instead of count.
Collision Type Use: Planes
World Collision is extremely expensive (raycasts per particle). Use Planes (simple geometric collision) whenever possible.
Render Mode Avoid: Mesh
Mesh Particles (3D models) are heavy on the vertex shader. Use Billboards (2D sprites facing camera) for 90% of effects.
Texture Sheet Animation Recommended
Use a flipbook texture (grid of frames) to add detail and motion to a single particle, rather than spawning more particles.
The Overdraw Killer: Large, semi-transparent particles covering the screen force the GPU to redraw pixels multiple times. This destroys fill rate. Keep particles small or opaque.
