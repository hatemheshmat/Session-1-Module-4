Reading Performance Metrics

FPS (Frames Per Second)
Target: 72 / 90 / 120
The current refresh rate. If this dips below the target, you are dropping frames.
If unstable, check CPU vs GPU times to find the bottleneck.
App GPU Time
Target: < 11ms (at 90Hz)
Time spent by the GPU to render a single frame.
High? Reduce resolution, shader complexity, or overdraw.
App CPU Time
Target: < 11ms (at 90Hz)
Time spent by the Main Thread (scripts, physics, draw call submission).
High? Optimize scripts, physics, or reduce draw calls.
Stale Frames
Target: 0
Number of frames that missed the display deadline in the last second.
Any value > 0 indicates visible judder/stuttering.
Thermal Level
Safe: 0 - 2
Device temperature status. Level 3+ triggers thermal throttling (reduced clock speeds).
Avoid sustained high load to prevent throttling.
Used Memory
Limit: ~5.5GB (Quest 2)
Total RAM usage. Exceeding the limit causes the OS to kill the app.
Monitor texture compression and asset unloading.
