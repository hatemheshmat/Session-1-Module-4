Build Settings Optimization

Scripting Backend: IL2CPP
Mono is fast to build but slow to run. IL2CPP converts C# to C++ and compiles to native machine code. It is significantly faster and required for the Quest Store.
Architecture: ARM64
Quest 2, 3, and Pro are 64-bit Android devices. Always select ARM64. Never build for x86 or ARMv7 (32-bit) as they are deprecated and slower.
Graphics API
Use Vulkan. It has lower CPU overhead than OpenGLES3, allowing for more draw calls and better multi-threading support.
