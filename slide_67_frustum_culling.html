Frustum Culling

Automatic Efficiency
Unity automatically skips rendering objects that are outside the camera's view cone (Frustum). This saves massive GPU power.
The Mechanism
It doesn't check every triangle. It checks the object's Bounding Box (AABB). If the box touches the frustum, the object is drawn.
The Bounding Box Trap
If you have a giant mesh (like a single terrain for the whole level), its bounding box is huge. Even if you only see a tiny corner, Unity renders the entire object.

Fix: Split large meshes into smaller chunks.
