Read/Write Enabled OFF
The Memory Killer. If enabled, Unity keeps a copy of the mesh in system memory AND GPU memory. Disable this for all static objects to save 50% RAM.
Mesh Compression High
Reduces the file size of the mesh in the build. Does NOT affect runtime memory or performance, but crucial for keeping APK size down.
Optimize Mesh ON
Reorders vertices for GPU cache efficiency. Always enable this for better rendering performance.
Generate Lightmap UVs Contextual
Only enable for static objects that will be lightmapped. Adds overhead to import time and mesh data size.
