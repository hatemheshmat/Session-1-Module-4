Static Batching

The Concept
Unity takes all objects marked as Static and combines their meshes into one large mesh at build time. This allows them to be drawn in a single batch (Draw Call).
The Requirement
Objects must never move, rotate, or scale during gameplay. Perfect for buildings, terrain, rocks, and walls.
Memory Trade-off
Static Batching creates a copy of the combined mesh in memory. If you have a dense forest of 1,000 identical trees, Static Batching will consume huge amounts of RAM. Use GPU Instancing for that instead.
