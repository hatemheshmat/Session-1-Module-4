Post-Processing Effects

Depth of Field / Motion Blur Very Expensive
Ambient Occlusion (SSAO) Expensive
Bloom (Fast Mode) Moderate
Color Grading (LUT) Cheap
Avoid These
Depth of Field and Motion Blur require multiple texture lookups per pixel. On a 4K VR display, this destroys fill-rate performance. Never use them on Quest.
Use with Caution
Bloom can be used if set to "Fast Mode" (fewer blur passes). Vignette is generally cheap and helps with motion sickness.
Recommended
Color Grading (Tonemapping) is the most efficient way to improve visuals. It uses a simple Lookup Table (LUT) to change colors with negligible performance cost.
Anti-Aliasing Note: Do NOT use Post-Process AA (SMAA/FXAA). Use MSAA 4x (Hardware AA) in Quality Settings. It is optimized for mobile GPUs and looks sharper in VR.
